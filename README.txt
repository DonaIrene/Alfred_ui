Configuração do directório de download:

1º - Abrir o Autolocate_downloads.pyw (Utilizar um IDLE)

2º - Alterar 'source' para o directorio atual de Downloads. (ex: c:\\users\\nome de utilizador\\Downloads\\) *nota, utilizar sempre barras duplas \\ *

3º - Alterar 'newSource' para o novo directório na qual deseja guardar os downloads. (ex: d:\\Downloads) *nota, não insira as duas ultimas \\ * 



Definir Arranque Automatico:

Mova o script Autolocate_downloads.pyw para a pasta de Arranque do Windows.
ex: C:\Users\User\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup
